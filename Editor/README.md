# Imventory Importer Jinxxy

## RU

Editor-инструмент для Unity, который помогает авторизоваться в Jinxxy, посмотреть свои inventory items и скачать файлы продукта прямо в проект.

Основное:
- Открытие через `Tools/Jinxxy/Open Window`.
- Вход через Jinxxy email/password или сохраненный аккаунт.
- Загрузка inventory с учетом пагинации.
- Просмотр файлов продукта через кнопку `Files`.
- Скачивание поддерживаемых файлов: `.zip` и `.unitypackage`.
- `.zip` распаковывается в `Assets/_DenisHik/JinxxyDownloader/zips/<product name>/`.
- `.unitypackage` скачивается в папку продукта и открывается для импорта в Unity.

## EN

Unity Editor tool for signing in to Jinxxy, browsing your inventory items, and downloading product files directly into the project.

Features:
- Open via `Tools/Jinxxy/Open Window`.
- Login with Jinxxy email/password or a saved account.
- Inventory loading with pagination support.
- Product file list via the `Files` button.
- Supported downloads: `.zip` and `.unitypackage`.
- `.zip` files are extracted into `Assets/_DenisHik/JinxxyDownloader/zips/<product name>/`.
- `.unitypackage` files are downloaded into the product folder and opened for Unity import.
